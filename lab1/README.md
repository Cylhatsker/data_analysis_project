# Data Analysis Project — Lab 1
Данный набор данных содержит информацию о популярности кофе и чая в разных странах.
### Состав показателей:
- column1 — целочисленный тип (int) — описание столбца  
- column2 — строковый тип (object) — описание столбца  
- column3 — вещественный тип (float) — описание столбца  
- column4 — дата (datetime) — описание столбца  
## Используемые библиотеки
В проекте были использованы следующие библиотеки:
- pandas — для работы с таблицами и анализа данных  
- matplotlib — для построения графиков  
- seaborn — для визуализации данных  
## Предобработка данных
В процессе анализа были выполнены следующие операции:
- Проверка типов данных:
  - метод `dtypes`
- Обнаружение пропусков:
  - метод `isnull()`
- Обработка пропусков:
  - метод `fillna()` (заполнение значений)
- Преобразование типов данных:
  - метод `astype()`
- Анализ статистических показателей:
  - метод `describe()`
## Визуализация данных
Были построены следующие графики:
- Гистограммы распределения значений  
- Boxplot (диаграмма размаха)  
- Тепловая карта корреляций (heatmap)
